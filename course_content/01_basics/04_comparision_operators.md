# Comparison Operators

Comparison operators compare two values and return `True` or `False`.

| Operator | Meaning |
|-----------|---------|
| == | Equal |
| != | Not Equal |
| > | Greater Than |
| < | Less Than |
| >= | Greater Than or Equal |
| <= | Less Than or Equal |

## Examples

```python
print(5 == 5)
```

```python
print(5 != 3)
```

```python
print(10 > 2)
```

```python
print(10 < 2)
```

```python
print(5 >= 5)
```

```python
print(4 <= 10)
```

## Comparing Strings

```python
print("apple" == "apple")
```

```python
print("apple" < "banana")
```
