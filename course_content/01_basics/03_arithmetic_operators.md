# Arithmetic Operators

Arithmetic operators perform mathematical calculations.

| Operator | Meaning |
|-----------|---------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| // | Floor Division |
| % | Modulus |
| ** | Exponent |

## Addition

```python
print(5 + 3)
```

## Subtraction

```python
print(10 - 4)
```

## Multiplication

```python
print(6 * 7)
```

## Division

```python
print(10 / 4)
```

## Floor Division

```python
print(10 // 4)
```

## Modulus

```python
print(10 % 3)
```

## Exponent

```python
print(2 ** 5)
```

## Operator Precedence

```python
print(2 + 3 * 4)
```

```python
print((2 + 3) * 4)
```
